Lucky Lotto Shaker

Random Number Generator

Documenter WebApp

The app is extremely simple. It has only one user interface window with nice looking spinner with the help of which you can select maximum random number to be generated. To use the app shake your device and when you feel vibration in your hand, five randmly generated numbers will appear on screen.

Reskin Ideas

1.    Change the Project Name to one that is relevant to your app

2.    Change Backgroud screen

3.    Change app logo

4.    Adjust translated string files to put your brand in text

5.    Add different number combinations

 

Import Project

1.    Open Android Studio

2.    Open Project->Locate your project->Select directory.

3.    At this step you should be ready to run your app on real device.

4.    Next you should change applicationId in "build.gradle" file from "com.kenzap_app" to your brand.
